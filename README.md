# Testing APIs with Mocha and SuperTest

Initialize the project.
```
npm init -y
```

Install the necessary packages
```
npm i --save-dev mocha supertest chai @babel/cli @babel/core @babel/node @babel/register @babel/preset-env dotenv
```
